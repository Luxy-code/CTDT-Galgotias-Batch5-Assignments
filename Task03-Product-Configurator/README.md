## 🚘 Product Configurator Details

### 1️⃣ Selected Tool & Product
- **Configurator Tool:** Tesla Design Studio (Web-based Product Configurator)
- **Product:** Tesla Model 3 Custom Vehicle

### 2️⃣ Process Flow Explanation
The Tesla Design Studio is a highly intuitive, consumer-facing product configurator that allows buyers to build their vehicle step-by-step. The flowchart illustrates the C-K (Concept-Knowledge) solution space exploration where the user navigates through specific decision nodes:

1. **Start:** User accesses the configurator tool online.
2. **Model & Powertrain:** User selects the vehicle base (e.g., Model 3) and determines the battery/motor configuration (Rear-Wheel Drive vs. Long Range AWD).
3. **Exterior Customization (Decision Node):** User decides whether to stick with the included stealth grey paint or explore the premium color solution space for an additional cost.
4. **Interior Customization:** User configures the wheels and the interior cabin styling.
5. **Software Customization (Decision Node):** A critical branch where the user decides whether to add the premium Full Self-Driving (FSD) capability.
6. **End:** The configurator generates the final unified product and price, leading to the order deposit.
